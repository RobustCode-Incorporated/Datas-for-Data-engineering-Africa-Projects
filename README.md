# 🌍 African Names Dataset Project

## 📌 Introduction

Avant de me lancer pleinement dans le **Data Engineering**, j’ai initié ce projet open source qui constitue une première étape importante : la création d’une base de données composée de **noms purement africains**.

L’objectif initial est de construire une fondation solide de données fiables et représentatives, qui pourra évoluer avec le temps. Nous prévoyons d’enrichir progressivement cette base avec de nouvelles informations afin de permettre des analyses de données pertinentes, réalistes et utiles dans différents contextes (statistiques, démographie, machine learning, etc.).

---

## 🎯 Objectif du projet

Ce projet vise à :

- Constituer une **base de données structurée de noms africains**
- Nettoyer et normaliser les données pour un usage analytique
- Préparer une base évolutive pour des projets de Data Engineering
- Permettre des analyses statistiques sur les noms et tendances
- Servir de fondation pour des cas d’usage en Data Science et IA

---

## 🧱 Structure actuelle des données

Les données sont organisées sous forme tabulaire :

| Nom | Post-nom | Prénom |
|-----|----------|--------|

Chaque entrée représente un individu avec une séparation normalisée des composants du nom.

---

## 🚀 Évolution du projet

Ce projet est conçu comme une base évolutive :

- Ajout progressif de nouvelles données
- Enrichissement avec des métadonnées (âge, région, sexe, etc.)
- Intégration possible avec des bases de données SQL
- Préparation pour des pipelines ETL (Extract, Transform, Load)
- Support futur pour des analyses avancées et modèles prédictifs

---

## 🛠️ Technologies utilisées

- Python (pandas, regex)
- CSV / format tabulaire
- Git / GitHub pour le versioning
- (Future) PostgreSQL / BigQuery / Spark

---

## 📊 Cas d’usage futurs
